CareerMatch — сайт помощи в выборе профессии
1. Цель сайта

Помочь пользователям подобрать профессии, которые соответствуют их интересам, навыкам, образованию, личностным типам (RIASEC/Holland) и желаемому уровню дохода.

Показывать рейтинг профессий, среднюю зарплату, востребованность и необходимые навыки.

Предоставлять визуализацию профиля пользователя и сравнение нескольких профессий.

Быстрое MVP с возможностью расширения коммерчески: подписка, курсы, интеграция с вакансиями.

2. Целевая аудитория

Студенты, выпускники

Люди, ищущие смену профессии

Молодые специалисты и карьерные ориентиры

HR и карьерные консультанты

3. Структура сайта
Главная страница

Логотип и название сайта

Краткое описание: «Подберём подходящую профессию с учётом ваших навыков и личностного типа»

Кнопка “Начать подбор” → переходит на анкету

Анкета / форма ввода

Поля анкеты:

Интересы: чекбоксы / мультивыбор (например: технологии, медицина, творчество, финансы, образование)

Навыки: текстовое поле с автодополнением или теги

Образование: селект (среднее, среднее спец., высшее, магистратура)

Желаемый доход: ползунок или поле ввода

Условия работы: чекбоксы (удалённая работа, офис, гибкий график, командировки)

RIASEC тест (Holland):

18–24 утверждения/вопроса (по 3–4 для каждого типа: R, I, A, S, E, C)

Пользователь оценивает согласие: шкала 1–5

Backend вычисляет профиль пользователя по Holland

Страница результатов подбора профессий

Список подходящих профессий с рейтингом соответствия (%)

Для каждой профессии:

Название

Средняя зарплата

Рынок вакансий (количество открытых вакансий)

Необходимые навыки

Тип работы (удалённая, офис, гибкий график)

Кнопка “Подробнее” для раскрытия информации

Фильтры и сортировка:

Мини-дашборд / визуализация

График RIASEC профиля пользователя

График средней зарплаты и востребованности профессий

Возможность сравнить 2–3 профессии

Цветовая визуализация совпадения профиля (тепловая карта или бары)

Дополнительные элементы UX/UI

Адаптивный дизайн для мобильных и десктопов

4. Логика работы и backend (Scala)

Подбор профессий:

Получение анкеты + RIASEC ответы

Вычисление профиля пользователя (R, I, A, S, E, C)

Вычисление совпадения профиля с профессиями из базы

Рейтинг и сортировка:

Совпадение RIASEC (например, корреляция профиля пользователя и профессии)

Совпадение навыков

Фильтр по зарплате и условиям работы

Вывод:

JSON с профессиями, рейтингом, зарплатой, навыками, типом работы

Агрегированные данные для графиков

5. База данных

Professions: название, навыки, RIASEC профиль, средняя зарплата, востребованность, тип работы

UserProfiles: временные данные анкет и результатов теста

Возможность расширять: интеграция с внешними API вакансий, курсов, статистики рынка

6. Docker и облако

Docker: контейнеризация backend (Scala API) + база данных

Docker-compose: для локальной разработки и быстрого деплоя

Облако: AWS/GCP/Azure для production

Масштабируемость при росте числа пользователей

Возможность развернуть мониторинг и логирование

7. Коммерческий потенциал

Интеграцию с вакансиями
